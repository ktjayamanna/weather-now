# Weather Now
Weather now is an alternative to the Non-Mac users who are looking for a weather app on web similar to the one on the Mac/iphone.

# Tech Stack
- Next.js 15
- TypeScript
- Tailwind CSS
- Zustand
- tRPC
- React Query
- Jest
- Cypress
- Vercel
- PostHog
